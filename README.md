# nodejs-hw — 01-express

Мінімальний бекенд на Express для роботи з колекцією нотаток (демо).

## Що реалізовано

- Express-сервер з middleware:
  - `express.json()` для роботи з JSON
  - `cors` для CORS
  - `pino-http` для логування запитів
- Маршрути:
  - `GET /notes` — повертає всі нотатки (поки що лише повідомлення)
  - `GET /notes/:noteId` — повертає одну нотатку за ID (повідомлення з ID)
  - `GET /test-error` — штучна помилка для перевірки error-middleware
- Middleware:
  - 404: `Route not found`
  - 500: повертає повідомлення про помилку (або безпечне повідомлення в production)

## Запуск локально

```bash
npm install
npm run dev
```

Сервер за замовчуванням стартує на порту з `.env` (`PORT`, стандартно 3000).

## Деплой

Для деплою на Render.com:
- створити сервіс типу `Web Service`
- вказати репозиторій і гілку `01-express`
- у налаштуваннях додати змінну оточення `PORT` (наприклад, `10000`)
- як стартову команду вказати:

```bash
npm install && npm run start
```
